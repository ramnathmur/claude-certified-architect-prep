01

What this is

A real course, not a cram sheet

Built on an AI‑first learning doctrine: understanding comes before exam tactics. You don't read documentation — you're taught, in dialogue, by an AI professor who leads with the “why,” and only then coached for the exam.

The live course is the CCA‑F Academy — a structured program run by two AI personas. A professor teaches every concept from an analogy and won't advance until you can explain it back in your own words; an exam‑coach runs comprehension checks, scenario drills, and timed mock exams. It spans 37 sessions across 7 phases (~29 hours), at your pace, with a persistent learner model that remembers where you're weak and re‑surfaces it via spaced repetition. Behind it sits a QA‑gated reference library — two faithful base courses plus nine architect‑depth extension lessons — that the system draws on for factual accuracy. It's reference, not assigned reading; the professor teaches everything. All of it is self‑contained and offline, with no paid API key.

03

Salient features

What's inside — and why each part earns its place

A judgment‑first program: it optimises for applied reasoning and cross‑domain synthesis (what the scaled exam over‑weights), not definition recall. Every exam fact is cited to the official study guide or Skilljar — never asserted from memory.

🎓

37‑session professor‑led program

Two AI personas: a professor who teaches each concept from an analogy, and an exam‑coach who runs drills, scenario cases, and timed mocks. Taught in dialogue across 7 phases — never a reading dump.

🧠

A learner model that remembers you

LEARNER-MODEL.md is persistent memory: mastery per concept on a state machine (Untested → Weak → Developing → Strong → Exam‑ready), a spaced‑repetition queue, and an interlink map for the cross‑domain synthesis the exam tests.

🎯

Genuine cold baseline + adaptive path

Session 1 measures you for real with a no‑hints diagnostic. Concepts you already explain cold get compressed, so your time goes to weak spots and held‑out mocks — the 37 sessions are a ceiling, not a floor.

📚

QA‑gated reference library

Two faithful base courses + nine architect‑depth, source‑cited extension lessons behind the Launchpad dashboard — the system's factual backbone, each shipped only after passing two QA gates.

📝

Weighted practice + sealed mocks

51 domain‑weighted questions and 3 integration checkpoints, plus a fresh 2×60 held‑out mock bank that stays sealed until mock time so readiness isn't over‑stated.

🗂️

Exam digest cram sheet

Knowledge areas mapped to the 5 domains with an anti‑pattern cheat sheet for the exam‑eve window — the exam's centre of gravity is spotting bad architecture.

🔬

Teach‑back gate — no fake progress

The professor advances only when you explain a concept back unprompted; “got it” doesn't count, an analogy that misses gets swapped for another, and synthetic/self‑driven runs never count toward mastery.

🚦

A Go/No‑Go readiness gate

Book the real exam only when a mock clears ≥ 720 with no domain or scenario below its floor — not on a passing total that hides a weak area.

The five exam domains

Where the weight is — and which phase teaches it

CCA‑F is scored on a scaled basis (pass = 720/1000): hard scenario and anti‑pattern items carry most of the points, so you study for judgment, not coverage. The exam draws 4 of 6 architecture scenarios per sitting. The Academy teaches in weight order and uses the official domain numbering.

DomainWeightTaught in
D1 · Agentic Architecture & Orchestration 27% Phase 1 · Sessions 2–9 (incl. hooks, escalation, session mgmt)
D3 · Claude Code Configuration & Workflows 20% Phase 2 · Sessions 10–15
D4 · Prompt Engineering & Structured Output 20% Phase 3 · Sessions 16–21
D2 · Tool Design & MCP Integration 18% Phase 4 · Sessions 22–26
D5 · Context Management & Reliability 15% Phase 5 · Sessions 27–31

A suggested 5‑week path

One session per sitting — don't double up

Active recall needs time to consolidate, so run one session per sitting (~2/week ≈ 5 weeks). A missed day moves the calendar, it doesn't delete a session — and the adaptive path compresses what you already know.

Week 4

Week 5

Remediation, two timed mocks, booking decision. Sessions 34–37.
